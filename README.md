# BatteryLogger-Android

Android app for continuous background battery data logging. Records voltage, 
current, temperature, and state to CSV for long-duration cycle analysis.

## Features
- Background logging service
- Records voltage, current, temperature, and battery state
- CSV output compatible with BatteryCharts-Android
- Configurable logging interval

## Requirements
- Android 5.0 (Lollipop) or later

## Installation
Download the APK from the [Releases](../../releases) page.  
Enable "Install from unknown sources" in Android settings before installing.

## Usage
1. Launch BatteryLogger
2. Set logging interval
3. Start logging — runs in background
4. CSV file saved to device storage for import into BatteryCharts
